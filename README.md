# Evaluasi-Tengah-Semester
Hasil Evaluasi Tengah Semester Membuat GUI Monitoring Hujan
Junian Akhas Natasaleh (2042251004), Ilham Fathirrahman (2042251096)
Untuk menjalankan program lakukan :
Masuk ke folder project
Tambahkan dependency pada file Cargo.toml
Buat program di file src/main.rs
Simpan seluruh perubahan file
Jalankan proses build program dengan ketik cargo build pada terminnal
Jalankan program GUI monitoring hujan dengan ketik cargo run pada terminnal
GUI monitoring hujan akan muncul
Untuk menghentikan program, tekan CTRL + C pada terminal atau tutup jendela GUI.
